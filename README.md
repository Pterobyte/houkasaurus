## ‚òïÔ∏è Getting Started

To get it running locally, install dependencies with yarn and run the commands

### Prerequisites

* yarn

```sh
npm install yarn@latest -g
```

### Installation

```sh
# Install dependencies
yarn install

# Fetch content from CMS backend
yarn fetch

# Run local dev server
yarn dev
```
